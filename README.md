# Learning Management System (LMS)

A comprehensive Learning Management System built using **React** and **Django** to streamline academic administration, student management, attendance tracking, assignments, assessments, and communication within educational institutions.

---

## Overview

This LMS was developed to digitize and simplify school and educational institution operations by providing a centralized platform for administrators, teachers, students, and parents.

The system enables efficient management of academic records, attendance, assignments, marks, behavioral logs, notices, and other day-to-day educational workflows.
